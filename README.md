# On-Boarding

This project is a React + Vite onboarding flow with role-based dashboards.

## Development

1. Install dependencies:
   `npm install`
2. Start the dev server:
   `npm run dev`

## Production build

Create the production bundle with:

`npm run build`

The built site is generated in `dist/`.

## GitHub Pages deployment

This repository uses GitHub Actions to deploy the Vite build output to GitHub Pages.

1. Push to `main` or `master`.
2. In GitHub, open `Settings > Pages`.
3. Set `Source` to `GitHub Actions`.
4. Wait for the `Deploy to GitHub Pages` workflow to finish.

The Vite base path is already configured for this repository name: `/On-Boarding/`.
